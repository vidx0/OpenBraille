How BrailleBoard works
01 — INPUT
Text arrives
Any text from a phone app, PC, or tablet sends characters via Bluetooth or USB serial to the Arduino Mega.
02 — TRANSLATE
Python engine converts
Our translation engine maps each character to its 6-dot Braille pattern. Grade 2 contractions compress common words.
03 — DISPLAY
Solenoids raise pins
36 solenoids (6 per cell × 6 cells) physically push pins up or down. The student reads by touch, then taps the advance button for the next 6 characters.
Built at a 24-hour hackathon  ·  SS4T themed  ·  Students with Visual Disabilities
Hardware: Arduino Mega + 36× solenoids + ULN2803 drivers + 3D-printed housing
Software: Python + liblouis + PySerial + React Native companion app
